■簡易Q&A

------------------------------------------------------------------------------
＜Q.1＞
  「index.html」にアクセスした時にローディング画面が終わらない。

＜A.1-1＞
　設置先のURLがサブディレクトリ（例：http://example.com/udonarium/）の場合、
　上手く動作しない可能性があります。

　「index.html」の<base href="./">を書き換えて、サーバのサブディレクトリ位置を
　設定してみてください。（例：<base href="/udonarium/">）
　
＜A.1-2＞
  「.htaccess」を削除してみてください。
  「gz圧縮ファイル」の配信制御が上手く動作していない可能性が考えられます。

＜A.1-3＞
  古いサーバーキャッシュが残っているのかもしれません。
　Ctrl + F5でブラウザをリロードしてみてください。

------------------------------------------------------------------------------
＜Q.2＞
  「index.html」に画面は表示されるが、PeerIDが「???」のままになる。

＜A.1＞
  SkyWay API Keyの設定が間違っている可能性があります。
  以下の項目について確認してみてください。

　　・SkyWay API Keyを取得し、「assets/config.yaml」に【API Key】を記述した。
　　・API Keyの「利用可能ドメイン名」に【あなたのサーバドメイン】を登録した。
　　・「APIキー認証」は【利用しない】に設定した。

------------------------------------------------------------------------------

＜A.EX＞
